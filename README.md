# Docker Image for [Siege](http://www.joedog.org/siege-home/)

## How to use.

### setup

```console
$ git clone https://github.com/sortigoza/docker-siege.git
$ cd docker-siege
$ docker-compose build
```

### help

```console
$ ./run
```

### Performance test

```console
$ ./run -d1 -r10 -c25 example.com
```
